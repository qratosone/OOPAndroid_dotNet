# 简单的HTTP图片下载程序


### 运行说明
程序运行的时候，输入网页地址，程序会自动访问网页，并分析里面的图片，将其下载到自己所在的目录下面，文件名按照数字顺序进行命名。

### 代码说明
程序主要包括三个部分：

- 爬取HTML页面
这部分使用了`HttpClient`的相关方法，向服务器发送`GET`请求，然后将收到的HTML源代码保存在字符串中

- 分析图片链接
这部分使用C#的`Regex`类，通过正则表达式搜索图片链接，并根据需要进行Parsing，把地址中缺少的`HTTP`或者`HTTPS`补全

- 下载图片
这部分使用`WebClient`类，直接调用`DownloadFile`方法进行下载
